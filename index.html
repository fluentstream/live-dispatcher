Live Dispatcher Examples

Each page provides different Live Dispatcher connections and examples.

You can see the traffic in real time by opening your browser developer console, navigating to the "network" tab,
selecting the open socket connection, and opening the "messages" tab. Once you've done this, simply perform the relevant
action on your FluentStream system and you will see the associated event occur within the active socket.

See the README for additional information, config, and troubleshooting.
